The Basics of HTML

Every chunk of code is surrounded like this:
(line 1) <p></p> | So that's how you code! Surround everything in the less than and greater than signs.
Alright. That just makes it easy, because it's hard for me to type those in. I need to type in 4 characters!

Have you made your repositry? Yes? Then let's type in it! Look around, you'll find a pencil. Click it, and type in <DOCTYPE HTML>.
That lets the browser know this is a website made out of HTML. Now type in <head> <title>(insert name of website)</title>
</head>. The head tag is where you put things that won't actually show on the website.
The title tag is what'll show up when someone bookmarks your website or has a tab open. (Notice how it says "Editing (name)/index.html")
Now do the same thing below the /head line. Type <body><p></p></body>. The p tag stands for paragraph. It's the basic text block!
That's all the basic stuff. Get ready to get advanced!

Basics of CSS

CSS, the best part of websites. This code will include coloring, image placement, id tags, fonts, ANIMATION, all that. Yay!
By the way, CSS stands for Cascading Style Spreadsheets, a title I do not care of and I don't know why.
CSS is complicated to understand, but same as HTML. First tag: id's. ID's aren't really "tags", they're more taggers.
So get an element, such as the <p> element. Now add "id = "(element name)". Now it should look like <p id= "example">.
So now that this element is id'd, we need to add the description. This is a large block of text describing an element. It looks like:
#example {
;
Now all the color tags should go under this id, like:
#example {
color: rgb (21,46,28);
;
That color tag defines the color of the element you id'd. So now the paragraph is colored! But what color? Well, that's where the RGB comes in.
It specifies the color of the element by values. 21 means there's 21 red value, 46 green, and 28 blue. Also, you can use color names such as royal purple,
aqua, blue, orange, wheat, and tomato. Also, the background-color tag works the same way the color tag works, only it tampers with the background, not the text or whatever.
Next up: Fonts! The basic font is Sans-Serif, plain and printed. This font is Serif. This font is "cursive." These fonts can give the extra "fancy" on the site.
These are typed just like the color tag : font-family: cursive, sans-serif;. Hey, there's a comma! That means "backup font." If someone doesn't have the "cursive" font. it'll be sans-serif instead. Advanced fonts come next in:
the Quest of the CSS Fonts!

Medium CSS

First, let's work on fonts. There are many more tags for CSS about fonts, and one is font-size. The attribute for this looks like:
font-size: 14px;
font-size: 1.2em - px = pixels. Easy enough. em = relative size to the basic font. Not as easy. So, under the "body" tag, you could put:
font-size: 1em; and the text would be "1.2 em" - that would make it 1.2 the size of normal text! And, there are 3 tags about font style. One is
font-style, where it determines if italic or not, (font-style: italic;) font-weight, for bold, (font-weight: bold;) and font-variant.
Font-variant makes lowercase letters uppercase (no vice-versa) and makes the uppercase letters smaller. Now, you can change the width and height of images by:
width: 200px;
height: 350px;
And if you only have one of the duo (only height or width) it will calculate a width (if you have height) or height. (if width)
