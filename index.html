    The Basics of HTML

    Every chunk of code is surrounded like this:
    (line 1) <p></p> | So that's how you code! Surround everything in the less than and greater than signs.
    Alright. That just makes it easy, because it's hard to type those in. But make sure that the closing tag (/p) has a slash right there.
    Have you made your repositry? Yes? Then let's type in it! Look around, you'll find a pencil. Click it, and type in <DOCTYPE HTML>.
    That lets the browser know this is a website made out of HTML. Now type in <head> <title>(insert name of website)</title>
    </head>. The head tag is where you put things that won't actually show on the website.
    The title tag is what'll show up when someone bookmarks your website or has a tab open. (Notice how it says "Editing (name)/index.html")
    Now do the same thing below the /head line. Type <body><p></p></body>. The p tag stands for paragraph. It's the basic text block!
